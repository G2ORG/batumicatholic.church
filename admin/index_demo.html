📖 Мануал контент-менеджера ✠ORDER OF CHRIST'S PILGRIMS✠
Для тех, кто добавляет контент. Всё по шагам, без технических знаний.

🔐 1. Вход

  1. Откройте batumicatholic.church/admin/
  2. Введите логин/пароль от координатора ✠ORDER OF CHRIST'S PILGRIMS✠

📰 2. Добавить новость

  1. Меню → «Новости» → «+ Новая»
  2. Вкладка 🇬🇪 ქართ. — заголовок и текст (обязательно)
  3. Вкладки EN и RU — переводы (опционально)
  4. Выберите категорию, дату, загрузите фото
  5. «Опубликовать» → сразу на сайте
    «Черновик» → сохранить позже

🕐 3. Расписание

  1. Меню → «Расписание»
  2. Изменяйте строки, нажмите «Сохранить»

🎙️ 4. Проповедь

  1. Меню → «Проповеди»
  2. Название (KA), дата, язык → нажмите зону загрузки
  3. «Загрузить»

📅 5. Событие

  1. Меню → «События»
  2. Название (KA), дата, время → «Добавить»
